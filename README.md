## v5 新增核心功能

### 🔪 单图多条识别
- 一张图片含多个面料标签时，**自动分割识别**，每条标签生成独立的表格行
- 分割策略（按优先级）：
  1. 条形码/噪声行 → 块分隔
  2. 连续空行 ≥ 2 → 块分隔
  3. 我方编号行（独立5位数字）→ 附加元数据
  4. 工厂名行（含"有限公司"等）→ 新块开始
  5. 货号/Art 行重复出现 → 新块边界

### 🤖 编号连续性推理
- 识别后自动对全表 C 列做连续性推理
- 若某行编号 OCR 未能识别，通过前后相邻行插值/递增/递减推断
- **推理编号视觉标注**：C 列显示橙色背景 + 🤖 图标 + Tooltip 说明
- 用户手动修改后自动清除推理标记，变为确认值

## 核心功能
- **OCR识别**：Tesseract.js 中英文，自动解析B~I列
- **单图多条**：`parseOCRMulti()` 分割多标签，每图可输出多行
- **编号推理**：`inferSequentialCodes()` 全表连续性推理，缺失编号自动补全
- **HLWG/HLFG**：1开头5位→HLWG，5开头5位→HLFG，自动判断
- **H列英寸**：公式 ROUND(M×100/2.54-2,0)/ROUND(M×100/2.54,0)" 只读自动计算
- **图片绑定**：每行显示对应缩略图（压缩后存储，节省流量）
- **多人同步**：每15秒轮询DB合并其他用户新增数据
- **历史记忆**：右上角时钟图标→时间层级→图片层级
- **手机端**：拍照/相册→OCR→保存→电脑端自动同步
