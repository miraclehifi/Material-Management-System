# 产品愿景

**最后更新**: 2026-06-02

---

## 产品定位

一分三科面料组的内部工具，用于替代手工抄写面料标签的低效流程。管理员拍一张面料标签图片，系统自动识别并结构化录入，多人实时共享同一数据库，最终一键导出 Excel 归档。

## 目标用户

面料组的管理员和库管人员，日常需要对面料样品进行编目和记录。

## 核心价值

1. **效率**：OCR 自动识别替代手动抄写，单条录入时间从数分钟降至数十秒
2. **协作**：多人同时录入，数据实时合并，无需等待轮流操作
3. **移动性**：手机端独立录入，现场即时录入无需回到电脑
4. **追溯**：按批次管理历史录入记录，可随时查阅和载入

## 成功指标

- 单条面料记录录入时间 < 60 秒（含拍照和 OCR 识别）
- 多人并发使用时数据无丢失
- 导出的 Excel 文件与实际面料标签信息一致率 > 95%

## 面料字段说明

| 列 | 字段名 | 说明 |
| -- | ------ | ---- |
| B | 面料描述 | 品名、颜色、花型等 |
| C | 编号 | HLWG（梭织）或 HLFG（针织）格式，如 HLWG11471 |
| D | 成分 Content | 纤维成分，如 60% Cotton 40% Polyester |
| E | 规格 Density | 纱支规格，如 32S×32S |
| F | 克重 | 单位 g/m²，如 180 |
| G | 幅宽(M) | 门幅（米），如 1.60 |
| H | 幅宽(英寸) | 由 G 列自动计算，格式如 61/63" |
| I | 工厂批号 | 工厂简称+货号，如 天强TM0815-198159 |
