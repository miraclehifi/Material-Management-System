# F-005 Excel导出 — Feature Scope

**创建方式**: adopt-scan 自动提取  
**最后更新**: 2026-06-02

---

## 目标

一键将当前所有面料数据导出为格式规范的 Excel 文件，供后续归档、打印或共享使用。

## 边界

**包含**：
- 导出当前表格全部行（B-I 列，共8个数据列）
- 固定表头（面料描述、编号、成分、规格、克重、幅宽M、幅宽英寸、工厂批号）
- 列宽预设
- 文件名含日期（前缀可配置）
- 工作表名称可配置
- 表格为空时提示不可导出

**不包含**：
- 导出选中行
- 导出图片
- 多 Sheet 导出
- 导出为 CSV

## 核心能力

1. 点击「导出 Excel」按钮，读取当前 `tableRows` 数据
2. 使用 SheetJS（xlsx.js）生成 .xlsx 文件并触发浏览器下载
3. 文件名格式：`{前缀}_{YYYYMMDD}.xlsx`

## 涉及端

| 端 | 覆盖能力 |
| -- | -------- |
| 桌面端 | 完整功能 |
| 手机端 | 不涉及 |

## 对外 Facade

无

## 关键实现位置

| 功能 | 文件 | 函数 |
| ---- | ---- | ---- |
| 导出入口 | js/app.js | `exportExcel()` |
