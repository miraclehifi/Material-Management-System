# F-007 系统设置 — Feature Scope

**创建方式**: adopt-scan 自动提取  
**最后更新**: 2026-06-02

---

## 目标

提供用户可调节的系统参数，适应不同使用场景（多人协作时降低同步间隔、调整图片大小减少存储压力等）。

## 边界

**包含**：
- Excel 导出设置（工作表名称、文件名前缀）
- 同步间隔设置（5-60秒）
- 缩略图最大尺寸设置（60-300px，影响图片存储大小）
- 成分缩写对照表预览（前15条）
- 设置保存（localStorage 持久化）
- 设置面板（右侧滑入侧边栏）

**不包含**：
- 成分缩写表的增删改（目前仅预览）
- 工厂关键词管理（在 OCR 设置区独立存在）
- 用户账号管理
- 数据库连接配置

## 核心能力

1. 点击「设置」按钮打开设置面板
2. 修改参数后点击「保存设置」，写入 localStorage，部分设置立即生效（如同步间隔重启定时器）

## 涉及端

| 端 | 覆盖能力 |
| -- | -------- |
| 桌面端 | 完整功能 |
| 手机端 | 不涉及 |

## 对外 Facade

无

## 关键实现位置

| 功能 | 文件 | 函数 |
| ---- | ---- | ---- |
| 打开/关闭设置 | js/app.js | `openSettings()`, `closeSettings()` |
| 保存设置 | js/app.js | `saveSettings()` |
| 配置读写 | js/app.js | `getCfg()`, `setCfg()` |
