# F-002 面料数据表格管理 — Feature Scope

**创建方式**: adopt-scan 自动提取  
**最后更新**: 2026-06-02

---

## 目标

提供类 Excel 的表格界面，让管理员能直接在浏览器中查看和编辑面料数据，所有改动实时保存到数据库。

## 边界

**包含**：
- 表格展示（9列：选框、图片、序号、B-I各字段）
- 单元格直接点击编辑（输入框 in-place editing）
- 新增空白行
- 删除选中行（支持多选）
- 全选/取消全选
- H 列自动计算（幅宽M → 英寸，只读公式列）
- C 列编号自动补全（输入5位数字 → 自动转为 HLWG/HLFG 格式）
- 推理编号特殊样式标注（橙色高亮 + tooltip）
- 单元格失焦后防抖保存（600ms debounce）
- 图片缩略图显示（点击可放大灯箱查看）
- 数据从数据库加载（页面初始化时拉取最新300条）
- 行计数展示

**不包含**：
- 列排序/筛选
- 分页（当前固定 limit=300）
- 批量编辑
- 撤销/重做
- 列宽调整

## 核心能力

1. 页面加载时自动从 `tables/fabric_records` 拉取数据并渲染
2. 任意单元格点击可直接编辑，失焦后自动保存到数据库
3. 支持选中多行后批量删除
4. G 列（幅宽M）输入后 H 列（英寸）实时联动计算
5. C 列（编号）输入5位数字自动转换为 HLWG/HLFG 编号格式

## 涉及端

| 端 | 覆盖能力 |
| -- | -------- |
| 桌面端 | 完整功能 |
| 手机端 | 不涉及（手机端为独立录入表单，见 F-006） |

## 对外 Facade

无

## 共享 Domain

- **fabric_records 数据表**：与 F-003（同步）、F-004（历史）、F-005（导出）、F-006（手机录入）共享同一数据源
